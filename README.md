# Viral Music Telegram Bot (Ready to Deploy)

This bot promotes a song virally on Telegram with:
- Referral tracking
- Automatic leaderboard
- Reward notifications

## Deployment on Railway

1. Sign up at https://railway.app/
2. Create a new project → Upload repo or connect GitHub
3. Add Environment Variable:
   - Key: BOT_TOKEN
   - Value: Your bot token from @BotFather
4. Deploy the project → Bot is live 24/7
5. Share bot username for users to start and get referral link

## Commands
/start - join bot & get song + referral link
/stats - check your referrals
/leaderboard - show top promoters
